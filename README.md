# Hand Gesture Volume Control using MediaPipe

## Description
Controls system volume using hand gestures captured via webcam using MediaPipe and OpenCV.

## Features
- Real-time hand tracking
- Gesture-based volume increase/decrease
- Works with webcam

## Tech Stack
- Python
- OpenCV
- MediaPipe
- Pycaw

## How to Run
1. Install dependencies
   pip install -r requirements.txt
2. Run
   python main.py
